# 🏙️ AI와 라즈베리파이를 활용하여 범죄를 예방하고 안전한 주거 환경을 설계하는 스마트 보안 도시 프로젝트

<br>
<br>

## 📌 프로젝트 한눈에 보기 :
2025 AI 지니어스 아카데미에서 ‘AI를 통한 사회문제 해결’을 주제로 진행한 스마트 보안 도시 기획 프로젝트입니다. 도시와 아파트 단지에서 발생하는 외부인 침입과 야간 범죄 문제를 해결하기 위해 AI 카메라, 순찰 로봇, 스마트 가로등, 중앙 관제 시스템이 연동되는 보안 환경을 설계했습니다. 라즈베리파이와 인공지능 기술을 CPTED 범죄예방환경설계에 결합하여, 기존의 정적인 범죄 예방 방식이 가진 실시간 대응의 한계를 보완하고자 했습니다.

<br>

## 🚨 문제 정의 :
도시와 아파트 단지에서는 외부인의 무단 침입, 폭행, 절도와 같은 범죄가 지속적으로 발생하고 있습니다. 특히 야간이나 관리 인력이 부족한 시간에는 위험 상황을 빠르게 발견하고 대응하기 어렵다는 문제가 있습니다.
기존 CPTED 방식은 조명, 공간 배치, 출입 통제와 같은 물리적 환경 개선에 집중되어 있어 범죄 예방에는 효과적이지만, 실제 위험 상황을 실시간으로 감지하고 능동적으로 대응하는 데에는 한계가 있습니다.
이에 AI와 IoT 기술을 활용하여 외부인 출입과 위험 상황을 자동으로 감지하고, 관련 정보를 중앙 관제실에 전달하는 스마트 보안 시스템을 구상했습니다.

<br>

## 💡 해결 아이디어 :
아파트 단지에 AI 카메라, 자율주행 순찰 로봇, 스마트 AI 카메라 가로등을 배치하고 각 장치에서 수집한 정보를 중앙 관제 시스템으로 전달하도록 설계했습니다.

ex) 등록된 입주민 얼굴 인식 → 자동 출입 허용  
미등록 외부인 감지 → 지속적인 감시 및 방문 목적 확인  
이상 행동·위험 음원 감지 → 중앙 관제실로 실시간 영상 전송  
사람 감지 → 스마트 가로등 자동 점등  
사람이 없는 경우 → 스마트 가로등 자동 소등

AI 카메라는 등록된 입주민과 미등록 외부인을 구분하며, 외부인이 감지되면 순찰 로봇이 방문 목적을 질문하도록 구성했습니다. 응답이 없거나 위험 상황이 확인되면 영상과 관련 정보를 중앙 관제실에 전달하고, 관리자가 최종 판단하여 대응하도록 설계했습니다.

스마트 가로등은 사람과 주변 밝기를 감지하여 자동으로 점등과 소등을 수행하고, 각 장치에는 태양광 에너지를 적용하여 친환경적으로 운영할 수 있도록 기획했습니다. <br>

<br>

## 🎯 프로젝트 목표 :
이 프로젝트는 기존 CPTED의 정적인 범죄 예방 방식을 보완하고, 인공지능을 활용해 위험 상황을 실시간으로 감지하고 대응할 수 있는 주거 보안 환경을 설계하는 것을 목표로 했습니다.
(입주민 및 외부인 구분 -> 외부인 행동과 위험 요소 감지 -> 중앙 관제 시스템으로 정보 전송 -> 관리자의 신속한 판단과 대응 -> 안전한 주거 환경 구축) <br>

<br>

## ⚙️ 기획 과정:
1. 사회문제 조사 및 주제 선정  
＞ 도시와 아파트 단지에서 발생하는 외부인 침입과 야간 범죄 사례를 조사했습니다. 기존 CPTED 방식의 장점과 실시간 대응의 한계를 분석한 뒤, AI를 결합한 스마트 보안 도시를 프로젝트 주제로 선정했습니다.

2. 전체 시스템 구조 설계  
＞ AI 카메라, 순찰 로봇, 자동 출입문, 스마트 AI 카메라 가로등의 정보를 중앙 관제 시스템으로 전달하는 구조를 설계했습니다. 각 장치가 독립적으로 동작하는 것이 아니라 상황 정보를 공유하도록 서비스 흐름을 구성했습니다.

3. 주요 기능 기획  
＞ 얼굴 인식을 통한 입주민 구분, 외부인 방문 목적 확인, 위험 움직임 및 음원 감지, 실시간 영상 전송, 사람 감지 기반 가로등 제어 기능을 기획했습니다.

4. 구현 및 실행계획 수립  
＞ 예비 아파트 단지를 구축한 뒤 중앙 관제 시스템, 스마트 가로등, 순찰 로봇을 단계적으로 구성하는 계획을 세웠습니다. 야간 환경과 위험 음원 등 다양한 상황을 시뮬레이션하여 시스템의 안전성과 완성도를 검토하도록 기획했습니다.

5. 기술적 한계와 윤리 문제 분석  
＞ 마스크 착용과 조명 변화로 인한 얼굴 인식 오류, 일상적인 행동이 위험 상황으로 잘못 판단될 가능성을 분석했습니다. 개인정보 수집 동의, 정보 보관 기간, 사생활 침해와 감시 기술의 남용 가능성도 함께 고려했습니다. <br>

<br>

## 🛠 활용 기술 :
- **Raspberry Pi** - 카메라와 센서 제어 및 각 장치의 데이터 처리
- **Python** - 인공지능 기능과 장치 제어 프로그램 구성
- **OpenCV** - 카메라 영상 처리 및 얼굴 인식
- **face_recognition** - 등록된 입주민과 외부인 얼굴 구분
- **MediaPipe** - 사람의 움직임과 행동 분석
- **조도 센서** - 주변 밝기에 따른 스마트 가로등 제어
- **인체 감지 센서** - 사람의 유무 감지 및 자동 점등·소등
- **소리 센서** - 비명과 충격음 등 위험 음원 감지
- **태양광 전원 시스템** - 장치의 친환경적인 전력 공급
- **CPTED** - 범죄예방환경설계를 기반으로 한 서비스 구조 기획 <br>

## ⚠️ 기술적 한계 및 윤리적 고려사항 :
- 마스크 착용이나 조명 변화에 따라 얼굴 인식 오류가 발생할 수 있습니다.
- 일상적인 행동이나 소리가 위험 상황으로 잘못 판단될 가능성이 있습니다.
- 입주민의 동의 없이 얼굴과 행동 정보를 수집해서는 안 됩니다.
- 수집된 개인정보는 보안 목적으로만 사용하고 장기간 보관하지 않도록 해야 합니다.
- 과도한 감시로 인한 사생활 침해와 감시 기술의 목적 외 사용을 방지해야 합니다.
- AI의 분석 결과만으로 대응을 결정하지 않고, 관리자가 최종 판단하도록 설계해야 합니다.

## 🌱 기대 효과 및 배운 점 :
- AI 카메라와 순찰 로봇이 위험 상황을 실시간으로 확인하고 중앙 관제실과 정보를 공유함으로써, 범죄 발생 전 예방과 신속한 대응이 가능한 주거 보안 환경을 제안했습니다.
- 얼굴 인식, 센서, 순찰 로봇과 같은 개별 기술을 하나의 중앙 관제 시스템으로 연결하면서, 여러 장치가 데이터를 주고받는 통합 시스템의 구조를 설계하는 경험을 했습니다.
- 인공지능 기술은 정확도뿐만 아니라 개인정보 보호, 사생활 침해, 정보 오남용과 같은 윤리적 문제를 함께 고려해야 한다는 점을 배웠습니다.
- 팀장으로서 프로젝트의 방향을 정하고 역할을 분담하며, 팀원들의 의견을 하나의 서비스 흐름으로 정리하는 협업 과정을 경험했습니다.
